This time, I wanted to change the type of game and opted for a 2D arcade-style game, using Unity again. I only created Level 1, and it lacks polished graphics.
The player must collect several objects within the level in order to unlock the end-of-level door and exit. Along the way, they will encounter monsters that can be defeated by shooting projectiles or avoided by jumping from platform to platform.
I also added a menu where you can load a game or create a new one.
